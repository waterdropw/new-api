POST /v1/embeddings

嵌入（Embeddings）

文本向量生成接口，兼容 OpenAI Embeddings 格式

接口说明

嵌入接口用于将文本转换为向量表示，常用于向量检索、语义搜索、相似度计算等场景。

返回的向量维度和精度取决于具体模型，请参考对应模型的官方说明。

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

请求体参数

参数 类型 必需 说明
model string 是 模型名称，如 text-embedding-3-large
input string | array 是 要嵌入的文本，可以是单个字符串或字符串数组
encoding_format string 否 返回格式，可选 float 或 base64，默认 float
dimensions integer 否 输出向量维度（仅部分模型支持）

请求示例

cURL
curl -X POST "https://your-base-url/v1/embeddings" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "text-embedding-3-large",
    "input": [
      "什么是向量嵌入？",
      "如何在生产环境中使用 Embeddings？"
    ],
    "encoding_format": "float"
  }'
