音频（Audio）接口总览

覆盖文本转语音、音频转录、音频翻译三类能力，兼容 OpenAI Audio 风格。

能力覆盖

建议按 endpoint 子页面阅读参数与示例，避免不同请求格式混淆。

能力 HTTP 路径 请求类型
文本转语音 POST /v1/audio/speech application/json
音频转录 POST /v1/audio/transcriptions multipart/form-data
音频翻译 POST /v1/audio/translations multipart/form-data

Endpoint 子页面

每个 endpoint 页面包含独立参数表、示例和错误响应。

🔊

文本转语音

POST /v1/audio/speech，支持在线测试与音频下载。
