POST /v1/audio/translations

音频翻译

将音频中的语音内容翻译为英文文本，使用 multipart/form-data 上传音频文件。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

请求参数

参数 类型 必需 说明
file file 是 音频文件，支持 mp3、mp4、mpeg、mpga、m4a、wav、webm
model string 是 模型名称，如 whisper-1
prompt string 否 引导翻译风格或术语偏好
response_format string 否 返回格式：json、text、srt、verbose_json、vtt
temperature number 否 采样温度，范围 0 - 1

请求示例

cURL
curl -X POST "https://your-base-url/v1/audio/translations" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -F file="@speech_zh.mp3" \
  -F model="whisper-1"

响应示例

200 OK
{
  "text": "Hello, welcome to the speech translation feature."
}

常见错误响应

400 Bad Request
{
  "error": {
    "message": "audio duration is too long",
    "type": "invalid_request_error",
    "param": "file",
    "code": "audio_too_long"
  }
}

在线测试说明

当前页面接口需要上传文件（multipart/form-data），不支持通过文档页内 JSON 表单直接测试。请使用 cURL、Postman 或 SDK 上传音频文件。
