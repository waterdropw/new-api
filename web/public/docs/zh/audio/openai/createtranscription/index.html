POST /v1/audio/transcriptions

音频转录

将音频内容转换为文本，使用 multipart/form-data 上传音频文件。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

请求参数

参数 类型 必需 说明
file file 是 音频文件，支持 mp3、mp4、mpeg、mpga、m4a、wav、webm
model string 是 模型名称，如 whisper-1
language string 否 音频语言（ISO-639-1），如 zh、en
response_format string 否 返回格式：json、text、srt、verbose_json、vtt
temperature number 否 采样温度，范围 0 - 1

请求示例

cURL
curl -X POST "https://your-base-url/v1/audio/transcriptions" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -F file="@audio.mp3" \
  -F model="whisper-1" \
  -F language="zh"

响应示例

200 OK
{
  "text": "你好，欢迎使用语音识别功能。"
}

常见错误响应

400 Bad Request
{
  "error": {
    "message": "unsupported audio format",
    "type": "invalid_request_error",
    "param": "file",
    "code": "invalid_audio_file"
  }
}

在线测试说明

当前页面接口需要上传文件（multipart/form-data），不支持通过文档页内 JSON 表单直接测试。请使用 cURL、Postman 或 SDK 上传音频文件。
