API 参考 / 图像 / 图像生成
POST /v1/images/generations

图像生成

根据文本提示生成图像，兼容 OpenAI Images Generations 接口。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

请求参数

参数 类型 必需 说明
model string 否 模型名称，如 gpt-image-1、dall-e-3
prompt string 是 图像描述文本
size string 否 输出尺寸，如 1024x1024、1792x1024、1024x1792
n integer 否 生成数量，默认 1
quality string 否 图像质量：standard 或 hd
response_format string 否 返回格式：url 或 b64_json

请求示例

cURL
curl -X POST "https://your-base-url/v1/images/generations" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "gpt-image-1",
    "prompt": "一幅带有星空和山峦的插画风景",
    "size": "1024x1024",
    "n": 1
  }'

响应示例

200 OK
{
  "created": 1700000000,
  "data": [
    {
      "url": "https://example.com/generated-image.png"
    }
  ]
}

常见错误响应

400 Bad Request
{
  "error": {
    "message": "invalid image size",
    "type": "invalid_request_error",
    "param": "size",
    "code": "invalid_size"
  }
}
