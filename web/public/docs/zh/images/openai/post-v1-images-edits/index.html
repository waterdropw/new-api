POST /v1/images/edits

图像编辑

基于输入图像与提示词进行重绘或局部编辑，使用 multipart/form-data 上传文件。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

请求参数

参数 类型 必需 说明
model string 是 模型名称，如 gpt-image-1
prompt string 是 编辑提示词，描述目标图像效果
image file 是 待编辑原图（png/jpg/webp）
mask file 否 可选透明蒙版，仅重绘透明区域
size string 否 输出尺寸，如 1024x1024

请求示例

cURL
curl -X POST "https://your-base-url/v1/images/edits" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -F model="gpt-image-1" \
  -F prompt="将背景替换为夜晚城市霓虹风格" \
  -F image="@source.png" \
  -F mask="@mask.png"

响应示例

200 OK
{
  "created": 1700000000,
  "data": [
    {
      "url": "https://example.com/edited-image.png"
    }
  ]
}

常见错误响应

400 Bad Request
{
  "error": {
    "message": "image is required",
    "type": "invalid_request_error",
    "param": "image",
    "code": "missing_image"
  }
}

在线测试说明
