图像（Images）接口总览

覆盖图像生成与图像编辑两类能力，兼容 OpenAI Images 接口格式。

能力覆盖

生成接口为 JSON 请求，编辑接口为 multipart 上传，建议分页面阅读。

能力 HTTP 路径 请求类型
图像生成 POST /v1/images/generations application/json
图像编辑 POST /v1/images/edits multipart/form-data

Endpoint 子页面

每个 endpoint 页面提供独立参数说明、错误响应与调用示例。
