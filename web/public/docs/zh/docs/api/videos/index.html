API 参考 / 视频
POST /v1/videos/generations

视频（Videos）

基于文本或图像生成短视频，或对现有视频进行编辑和变换

接口说明

视频接口用于基于文本或图像生成短视频，或对现有视频进行编辑和变换。

视频生成功能依赖具体模型支持，可用模型请参考控制台配置。

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

请求体参数

参数 类型 必需 说明
model string 是 模型名称
prompt string 是 视频描述文本
size string 否 视频分辨率，如 1280x720、1920x1080
duration integer 否 视频时长（秒）
image string 否 参考图像 URL（图生视频场景）

请求示例

cURL
curl -X POST "https://your-base-url/v1/videos/generations" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "video-model",
    "prompt": "一只猫在草地上奔跑的慢动作视频",
    "size": "1280x720"
  }'

响应示例

200 OK
{
  "created": 1700000000,
  "data": [
    {
      "url": "https://example.com/generated-video.mp4",
      "revised_prompt": "一只猫在草地上奔跑的慢动作视频..."
    }
  ]
}
