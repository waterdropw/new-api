POST /v1beta/models/{model}:generateContent

Gemini文本聊天

代理 Gemini API 请求，支持 Gemini 2.5 Pro 等模型

路径格式：/v1beta/models/{model_name}:{action}
示例：/v1beta/models/gemini-2.5-pro:generateContent、 /v1beta/models/gemini-2.5-pro:streamGenerateContent?alt=sse

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

路径参数

参数 类型 必需 说明
model string 是 模型名称，如 gemini-2.5-pro

请求体参数

参数 类型 必需 说明
contents array<object> 否 对话内容列表
generationConfig object 否 生成配置（温度、topP、maxOutputTokens 等）
safetySettings array<object> 否 安全设置
tools array<object> 否 工具配置
systemInstruction object 否 系统指令

请求示例

cURL
curl -X POST "https://your-base-url/v1beta/models/gemini-2.5-pro:generateContent" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [{ "text": "你好，请介绍一下自己" }]
      }
    ],
    "generationConfig": {
      "temperature": 0.7,
      "maxOutputTokens": 2048
    }
  }'

响应示例

200 OK
{
  "candidates": [
    {
      "content": {

      "safetyRatings": []
    }
  ],
  "usageMetadata": {
    "promptTokenCount": 10,
    "candidatesTokenCount": 20,
    "totalTokenCount": 30
  }
}
