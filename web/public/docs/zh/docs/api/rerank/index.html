重排序（Rerank）

文档重排序 / 排名接口，优化向量检索结果

接口说明

重排序接口用于对一组文档根据查询语句进行相关性排序，常用于向量检索后的 rerank 流程。

请求示例

Request Body
{
  "model": "bge-reranker-v2-m3",
  "query": "什么是 New API？",
  "documents": [
    "New API 是一个 AI 模型网关...",
    "API 是应用程序接口的缩写...",
    "New API 支持多种模型..."
  ],
  "top_n": 3
}
