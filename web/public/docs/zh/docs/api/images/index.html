POST /v1/images/generations

图像（Images）

AI 图像生成与编辑接口，兼容 OpenAI Images 格式

接口说明

图像接口用于生成或编辑图像，兼容 OpenAI /images/generations 等接口的请求格式。

实际可支持的模型、分辨率和输出格式以控制台与上游模型文档为准。

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

请求体参数

参数 类型 必需 说明
model string 否 模型名称，如 gpt-image-1、dall-e-3，默认 dall-e-2
prompt string 是 图像描述文本，最大 4000 字符（dall-e-3）或 1000 字符（dall-e-2）
n integer 否 生成图像数量，默认 1（dall-e-3 仅支持 1）
size string 否 图像尺寸：256x256、512x512、1024x1024、1792x1024、1024x1792
quality string 否 图像质量：standard 或 hd，默认 standard
response_format string 否 返回格式：url 或 b64_json，默认 url
style string 否 图像风格：vivid 或 natural，默认 vivid（仅 dall-e-3）

请求示例

cURL
curl -X POST "https://your-base-url/v1/images/generations" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "gpt-image-1",
    "prompt": "一幅带有星空和山峦的插画风景",
    "size": "1024x1024",
    "n": 1,
    "quality": "standard"
  }'

响应示例

200 OK
{
  "created": 1700000000,
  "data": [
    {
      "url": "https://example.com/generated-image.png",
      "revised_prompt": "一幅带有星空和山峦的插画风景..."
    }
  ]
}
