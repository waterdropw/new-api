API 参考 / 视频 / 创建视频任务
POST /v1/videos/generations

创建视频任务

提交文生视频或图生视频请求，返回异步任务 ID。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

请求参数

参数 类型 必需 说明
model string 是 视频模型名称
prompt string 是 视频描述文本
size string 否 视频分辨率，如 1280x720、1920x1080
duration integer 否 视频时长（秒）
image string 否 参考图像 URL（图生视频）

请求示例

cURL
curl -X POST "https://your-base-url/v1/videos/generations" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "video-model",
    "prompt": "一只猫在草地上奔跑的慢动作视频",
    "size": "1280x720"
  }'

响应示例

200 OK
{
  "id": "video_task_abc123",
  "status": "queued",
  "created_at": 1700000000,
  "model": "video-model"
}

常见错误响应

429 Too Many Requests
{
  "error": {
    "message": "rate limit exceeded",
    "type": "rate_limit_error",
    "param": null,
    "code": "rate_limit_exceeded"
  }
}
