API 参考 / 视频 / 查询视频任务
GET /v1/videos/{task_id}

查询视频任务

查询视频异步任务状态、进度和结果元信息。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

路径参数

参数 类型 必需 说明
task_id string 是 视频任务 ID（创建接口返回）

请求示例

cURL
curl -X GET "https://your-base-url/v1/videos/video_task_abc123" \
  -H "Authorization: Bearer sk-xxxxxx"

响应示例

200 OK
{
  "id": "video_task_abc123",
  "status": "succeeded",
  "progress": 100,
  "created_at": 1700000000,
  "completed_at": 1700000123,
  "model": "video-model"
}

常见错误响应

404 Not Found
{
  "error": {
    "message": "video task not found",
    "type": "invalid_request_error",
    "param": "task_id",
    "code": "not_found"
  }
}

在线测试说明

该接口为 GET 请求且需动态替换 {task_id}，建议通过 cURL 或 SDK 调用。可先在创建视频接口获取任务 ID 再查询。

相关接口

POST /v1/videos/generations 创建视频任务 GET /v1/videos/{task_id}/content 下载视频内容
