API 参考 / 视频 / 获取视频内容
GET /v1/videos/{task_id}/content

获取视频内容

下载视频任务产出的二进制内容流。

认证

使用 Bearer Token 认证：Authorization: Bearer sk-xxxxxx

路径参数

参数 类型 必需 说明
task_id string 是 视频任务 ID（状态为 succeeded 后可下载）

请求示例

cURL
curl -L "https://your-base-url/v1/videos/video_task_abc123/content" \
  -H "Authorization: Bearer sk-xxxxxx" \
  --output output.mp4

响应说明

成功响应通常为视频流（如 video/mp4），建议直接保存文件或在播放器中播放。

常见错误响应

409 Conflict
{
  "error": {
    "message": "video task is not finished",
    "type": "invalid_request_error",
    "param": "task_id",
    "code": "task_not_ready"
  }
}

在线测试说明

该接口返回二进制内容且需动态替换 {task_id}，建议通过 cURL 或 SDK 调用并保存为文件。
