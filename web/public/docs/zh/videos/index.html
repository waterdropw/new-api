视频（Videos）接口总览

覆盖视频任务创建、状态查询、内容下载三个阶段，兼容 Sora 风格路径。

能力覆盖

视频接口是典型异步任务链路，建议按阶段阅读对应 endpoint 页面。

阶段 HTTP 路径 说明
创建任务 POST /v1/videos / /v1/videos/generations 提交文生视频或图生视频请求
查询状态 GET /v1/videos/{task_id} 轮询任务状态与进度
获取内容 GET /v1/videos/{task_id}/content 下载最终视频流

Endpoint 子页面

每个 endpoint 页面独立维护参数、示例和错误响应。
