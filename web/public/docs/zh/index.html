API 参考

面向 AI 模型调用者的 API 参考文档。兼容 OpenAI、Anthropic、Gemini 等多种格式。

快速开始

1
注册账号
创建一个账号用于登录控制台，可查看API调用情况、统计等信息。
2
创建令牌
登录控制台，创建 API 令牌。
3
开始调用
使用令牌调用 API 接口。

如果你已有兼容 OpenAI API 的调用代码，只需将 base_url 指向当前 API 服务地址，并配置对应的令牌，即可直接接入。
