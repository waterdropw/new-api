聊天（Chat）

根据对话历史创建模型响应，兼容 OpenAI Chat Completions 格式

能力覆盖

聊天接口支持 OpenAI / Anthropic / Gemini 三类协议风格。

协议风格 典型路径 适用场景
OpenAI Chat/Responses /v1/chat/completions / /v1/responses 多数 SDK 默认支持，迁移成本低
Anthropic Messages /v1/messages Claude 原生请求结构
Gemini GenerateContent /v1beta/models/{model}:generateContent Gemini 多模态与原生参数
