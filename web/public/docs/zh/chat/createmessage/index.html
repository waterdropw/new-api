POST /v1/messages

原生Claude格式

Anthropic Claude Messages API 格式的请求

需要在请求头中包含 anthropic-version。

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

请求头参数

参数 类型 必需 说明
anthropic-version string 是 Anthropic API 版本，如 2023-06-01
x-api-key string 否 Anthropic API Key（可选，也可使用 Bearer Token）

请求体参数

参数 类型 必需 说明
model string 是 模型名称，如 claude-3-opus-20240229
messages array<object> 是 对话消息列表
max_tokens integer 是 最大生成 token 数
system string | array 否 系统提示词
temperature number 否 采样温度 (0-1)
stream boolean 否 是否流式响应

请求示例

cURL
curl -X POST "https://your-base-url/v1/messages" \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "claude-3-opus-20240229",
    "messages": [
      {
        "role": "user",
        "content": "你好，请介绍一下自己"
      }
    ],
    "max_tokens": 1024
  }'

200 OK
