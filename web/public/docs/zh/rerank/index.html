重排序（Rerank）

文档重排序 / 排名接口，优化向量检索结果

接口说明

重排序接口用于对一组文档根据查询语句进行相关性排序，常用于向量检索后的 rerank 流程。

认证

使用 Bearer Token 认证，格式：Authorization: Bearer sk-xxxxxx

请求体参数

参数 类型 必需 说明
model string 是 模型名称，如 bge-reranker-v2-m3
query string 是 查询语句
documents array<string> 是 待排序的文档列表
top_n integer 否 返回前 N 个结果，默认返回全部
return_documents boolean 否 是否在结果中返回文档内容，默认 false

请求示例

cURL
curl -X POST "https://your-base-url/v1/rerank" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-xxxxxx" \
  -d '{
    "model": "bge-reranker-v2-m3",
    "query": "什么是大语言模型？",
    "documents": [
      "大语言模型是基于深度学习的自然语言处理模型，能够理解和生成人类语言。",
      "API 是应用程序接口的缩写，用于软件之间的通信。",
      "常见的大语言模型包括 GPT、Claude、Gemini 等。"
    ],
    "top_n": 3
  }'
